# PromptForge Product

## Promise

PromptForge Studio is a visual-world generation engine. It turns rough ideas into cinematic art directions, premium landing previews, Codex-ready prompts and anti-template creative systems.

## Target Users

- founders using AI to ship premium websites
- developers who want stronger art direction before implementation
- indie builders who need a concrete Codex prompt system
- service providers selling productized premium web direction

## Generated Blueprint Model

The Studio produces:

- headline
- subheadline
- CTA label
- selected visual world
- art direction summary
- benefits
- page sections
- Codex implementation prompt
- AGENTS.md rules
- review prompt
- polish prompt

## MVP Scope

This pass is frontend-only and deterministic. It does not call an API, execute user code or generate raw HTML.

## Out Of Scope

- backend persistence
- payments
- authentication
- live AI generation
- real 3D scene rendering
