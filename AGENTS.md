# PromptForge Studio — Agent Instructions

This project is a premium frontend landing page.

The goal is not to generate a generic SaaS or Tailwind template.
The goal is to create a premium 2026 animated landing page that feels like MotionSites, Awwwards, luxury digital studios, AI design labs, and cinematic product launches.

## Mandatory quality bar

Every UI must feel:
- premium
- original
- cinematic
- immersive
- high contrast
- motion-first
- conversion-oriented
- technically clean
- mobile-first

## Forbidden design output

Avoid:
- generic SaaS layouts
- plain centered hero sections
- boring cards
- default Tailwind look
- cheap glassmorphism
- random decorative blobs
- basic fade-up-only animation
- predictable navbar + hero + cards structure

## Required visual ingredients

Use:
- animated gradient mesh
- cursor-reactive spotlight
- floating prompt cards
- kinetic typography
- subtle grid overlay
- noise texture
- parallax depth layers
- premium CTA motion
- editorial oversized typography

## Technical rules

Use:
- React
- TypeScript
- Vite
- Tailwind CSS
- lucide-react
- semantic HTML
- accessible buttons
- reduced motion support
- mobile-first responsive layout

Avoid:
- uncontrolled React state updates on every animation frame
- heavy layout-triggering animation
- inaccessible interactive elements
- unnecessary dependencies
